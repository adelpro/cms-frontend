# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Migration from Next.js to Angular 20
- Asset management system with gallery view
- Publisher management portal
- Content standards documentation
- Multi-language support (English & Arabic)
- Authentication system (login, register, profile completion)
- Netlify deployment configuration for 3 environments
- Code formatting with Prettier
- Pre-commit hooks with Husky
- Comprehensive project documentation

### Changed
- Frontend framework from Next.js to Angular 20
- Styling system to LESS
- UI components to Ng-Zorro (Ant Design for Angular)

### Tech Stack
- Angular 20.3.7
- TypeScript 5.9.2
- Ng-Zorro (Ant Design) 20.3.1
- RxJS 7.8.0
- @ngx-translate for i18n
